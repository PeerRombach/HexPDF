HexPDF is a simple Java class making it easier to use Apache PDFBox for creating pdf documents from your Java application or web-service. HexPDF adds stuff like automatic page adding, word-wrap, newline awareness, left/right/center text alignment, table creation and image insertion.

View the Project on GitHub heksemann/HexPDF

Make it easy to generate pdf documents using Apache PDFBox as baseline. Convenience functions for text, table, images and footes.
